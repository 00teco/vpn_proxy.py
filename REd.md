# 🔒 GitHub VPN Proxy

**VPN de nível de aplicativo que funciona SEM permissão de administrador!**

Intercepte e analise o tráfego de aplicativos específicos como uma VPN, mas sem precisar de root ou admin. Perfeito para ambientes corporativos, escolas ou qualquer lugar onde você não tem privilégios administrativos.

## ✨ Características

- 🚫 **Sem admin/root** - Roda 100% em userspace
- 🎯 **Por aplicativo** - Intercepte apenas os apps que você escolher
- 🔐 **SOCKS5 proxy** - Protocolo padrão suportado por milhares de apps
- 🌐 **Tunelamento completo** - Funciona como uma VPN real para apps configurados
- 📊 **Logging completo** - Analise todo o tráfego em JSON
- 🚀 **Auto-configuração** - Lança Chrome/Terminal já configurados
- ⚡ **Zero dependências** - Apenas Python padrão (3.6+)
